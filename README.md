## About Me

I'm a Frontend Developer from Karachi, Pakistan, with practical experience building responsive, user-focused web interfaces using **React**, **JavaScript**, **HTML**, and **CSS**. I completed a professional software development diploma at **Aptech** and worked as a Frontend Developer before transitioning into independent growth.

I'm currently deepening my expertise across the **MERN Stack** — MongoDB, Express.js, React, and Node.js — with the goal of delivering complete, end-to-end web solutions rather than frontend work alone.

- 🔭 Currently building full-stack projects using the MERN Stack

- 🎯 Working toward long-term remote employment and freelance client work
- 🤝 Open to collaboration on real-world projects and long-term contracts
